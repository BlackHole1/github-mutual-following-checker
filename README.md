# github-mutual-following-checker

## Install

```shell
$ git clone git@github.com:BlackHole1/github-mutual-following-checker.git
```

## Usage

```shell
$ node ./index.js --token=GITHUB_TOKEN USER_NAME
# or
GITHUB_TOKEN=X GITHUB_USER_NAME=Y node ./index.js
```
